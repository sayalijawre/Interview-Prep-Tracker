# InterviewPrepTracker - AI-Powered Tech Practice Coach

An interactive practice tool that generates technology questions based on your 
skill level and provides AI-powered feedback on your answers.

## Features
- 🎯 Practice mode with AI-generated questions
- 📊 Real-time feedback and ratings
- 🎓 Multiple difficulty levels
- 💡 Continuous practice sessions
- 📈 Session summaries

## Tech Stack
- Frontend: React, Material-UI
- Backend: FastAPI, SQLAlchemy
- AI: OpenAI GPT-3.5-turbo
- Database: SQLite
